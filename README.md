# korean-hate-speech-koelectra

Bias, Hate classification with KoELECTRA

## Dataset

AI Challenge 2020의 데이터가 아닌 [korean-hate-speech](https://github.com/kocohub/korean-hate-speech)의 데이터를 사용하였습니다.

|          | # of data |
| -------- | --------: |
| train    |     7,896 |
| validate |       471 |
| test     |       974 |
